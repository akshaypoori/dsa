Data Structures and Algorithms

Arrays and Strings

Fundamental data structures. Understand common operations, manipulations, and related problems.

LeetCode - Array Problems | LeetCode - String Problems

Linked Lists

Linear data structures where elements are linked. Know different types and their applications.

LeetCode - Linked List Problems

Stacks and Queues

Abstract data types following specific principles (LIFO and FIFO). Understand their uses.

LeetCode - Stack Problems | LeetCode - Queue Problems

Trees

Hierarchical data structures. Focus on Binary Trees, BSTs, and related algorithms (traversals, search).

LeetCode - Tree Problems

Graphs

Representing relationships between objects. Learn about graph representations and traversal algorithms (BFS, DFS).

LeetCode - Graph Problems

Sorting and Searching Algorithms

Understand various sorting algorithms (Merge Sort, Quick Sort, Heap Sort) and searching algorithms (Binary Search).

Educative - Sorting & Searching

Dynamic Programming

Solving problems by breaking them down into overlapping subproblems. Practice common DP patterns.

LeetCode - Dynamic Programming Problems

Greedy Algorithms

Making locally optimal choices at each step with the hope of finding a global optimum.

Educative - Greedy Algorithms

Scalability

Designing systems that can handle increasing load and user traffic. Understand horizontal and vertical scaling.

Caching

Techniques to improve performance by storing frequently accessed data. Understand different caching strategies.

System Design Primer - Caching

Message Queues

AWS SQS | Apache Kafka

Architectural style where an application is composed of small, independent services.

Martin Fowler - Microservices

API Design

RESTful API | GraphQL
